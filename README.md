BLUEPRINT / RENCANA KERJA APLIKASI MOBILE (REVISI)
1. Nama Aplikasi
CampusBuddy – Aplikasi Asisten Mahasiswa
2. Latar Belakang
Mahasiswa sering menghadapi berbagai aktivitas akademik yang padat seperti pengelolaan tugas, pencatatan materi, pengaturan jadwal kuliah, serta pengelolaan keuangan sehari-hari. Permasalahan yang umum terjadi adalah kurangnya manajemen waktu, catatan yang tidak terorganisir, jadwal yang sering terlupakan, serta pengeluaran yang tidak terkontrol.
Berdasarkan permasalahan tersebut, dikembangkan aplikasi mobile berbasis Flutter yang mampu membantu mahasiswa dalam mengelola berbagai kebutuhan akademik dan aktivitas harian dalam satu platform terpadu.
3. Tujuan Pengembangan Aplikasi
Tujuan dari pengembangan aplikasi ini adalah:
•	Membantu mahasiswa mengelola tugas dan deadline
•	Memudahkan pencatatan materi kuliah
•	Membantu mengatur jadwal kegiatan dengan pengingat otomatis
•	Mengontrol pengeluaran harian mahasiswa
•	Menyediakan aplikasi multifungsi yang praktis dan efisien
4. Target Pengguna
•	Mahasiswa aktif
•	Pelajar tingkat akhir
•	Pengguna dengan aktivitas harian padat
5. Fitur Utama Aplikasi
5.1 Manajemen Tugas
•	Menambahkan tugas
•	Menentukan deadline
•	Edit dan hapus tugas
•	Menandai tugas selesai/belum
5.2 Scan & Catatan
•	Mengambil gambar catatan
•	Menyimpan catatan dalam bentuk foto
•	Menambahkan judul dan deskripsi
5.3 Manajemen Keuangan
•	Menambahkan pengeluaran
•	Kategori pengeluaran
•	Menampilkan total pengeluaran
•	Ringkasan keuangan
5.4 Jadwal & Notifikasi (FITUR BARU)
•	Menambahkan jadwal kegiatan (kuliah/kegiatan lain)
•	Menentukan hari dan waktu
•	Menampilkan jadwal harian
•	Edit dan hapus jadwal
•	Notifikasi pengingat otomatis sebelum kegiatan dimulai
5.5 Dashboard (Home)
•	Ringkasan:
o	Tugas terdekat
o	Jadwal hari ini
o	Catatan terbaru
o	Pengeluaran hari ini
5.6 Profil Pengguna
•	Informasi pengguna
•	Pengaturan aplikasi
6. Alur Penggunaan Aplikasi
1.	Pengguna membuka aplikasi (Splash Screen)
2.	Masuk ke halaman utama (Dashboard)
3.	Memilih menu:
o	Tugas
o	Scan Catatan
o	Keuangan
o	Jadwal
4.	Melakukan input data sesuai kebutuhan
5.	Data disimpan dan ditampilkan kembali
6.	Sistem memberikan notifikasi sesuai jadwal
7. Desain Antarmuka (UI/UX)
Desain aplikasi menggunakan konsep:
•	Minimalis dan modern
•	Menggunakan Material Design
•	Navigasi menggunakan Bottom Navigation Bar
Menu utama:
•	Home
•	Tugas
•	Scan
•	Keuangan
•	Jadwal
•	Profil
8. Teknologi yang Digunakan
•	Framework: Flutter
•	Bahasa Pemrograman: Dart
•	Database:
o	SQLite (lokal)
•	Notifikasi: flutter_local_notifications
•	Version Control: Git
•	Repository: GitHub
9. Output yang Dihasilkan
•	Aplikasi mobile berbasis Flutter
•	File APK
•	Repository GitHub
•	Dokumentasi README.md
•	Aplikasi terdaftar di Play Store
10. Penutup
Aplikasi CampusBuddy diharapkan dapat menjadi solusi praktis bagi mahasiswa dalam mengelola aktivitas akademik dan kehidupan sehari-hari secara lebih terorganisir, efisien, dan terintegrasi dalam satu aplikasi.
